# Sistema de votacion en Java

### Descripcion
Porgrama en Java que simula un sistema de votaciones en una urna, 
funciona ingresando primero el numero de candidatos elegidos, 
despues ingresando el numero de el candidato para votar por el,
hace un calculo del porcentaje de votos de cada candidato al terminar la votaciion
con el numero "0"


## Características

- Registro dinámico de votos
- Implementación con Programación Orientada a Objetos
- Cálculo automático de porcentajes
- Validación básica de candidatos
